# AFEngine

一个基于C++的现代游戏引擎练习项目

## 简介

AFEngine是一个使用C++编写的现代化游戏引擎项目，旨在提供一个功能完整的游戏开发环境。该项目包含了游戏引擎的核心组件，包括渲染系统、场景管理、物理系统、资源加载等模块。

## 项目结构

AFEngine/
├── AFEngine/ # 引擎核心代码
│ └── src/AF/
│ ├── Core/ # 核心系统（应用、窗口、输入等）
│ ├── Events/ # 事件系统
│ ├── ImGui/ # ImGui集成
│ ├── Loader/ # 资源加载器（如Assimp模型加载）
│ ├── Math/ # 数学库
│ ├── Physics/ # 物理系统
│ ├── Project/ # 项目管理
│ ├── Renderer/ # 渲染系统
│ ├── Scene/ # 场景管理系统和组件
│ └── Utils/ # 工具类
├── AF-Editor/ # 编辑器应用
└── Sandbox/ # 示例和测试项目

## 主要特性

### 渲染系统
- 基于OpenGL的渲染后端
- 支持可编程着色器管线
- 材质系统和纹理支持
- PBR（基于物理的渲染）支持
- 后期处理效果
- 多种几何体渲染（方框、圆形等）

### 场景管理系统
- 基于Entity-Component-System (ECS) 架构
- 支持多种组件类型：
  - Transform组件（变换）
  - SpriteRenderer组件（精灵渲染）
  - CircleRenderer组件（圆形渲染）
  - Camera组件（相机）
  - Mesh组件（网格）
  - Material组件（材质）
- 场景序列化和反序列化

### 编辑器功能
- 基于ImGui的可视化编辑器界面
- 场景层次面板
- 内容浏览器
- 可视化编辑工具（移动、旋转、缩放）
- 实时预览功能
- 项目和场景管理（新建、打开、保存）

### 资源管理
- 纹理加载和管理
- 模型加载（通过Assimp库支持多种格式）
- 着色器管理
- 材质系统

### 工具和辅助功能
- 性能分析工具
- 日志系统
- 输入处理系统
- UUID系统用于实体标识

## 构建说明

项目使用Premake5作为构建系统配置工具。

### 依赖项
- OpenGL
- GLFW (窗口和输入)
- GLAD (OpenGL函数加载)
- ImGui (界面)
- GLM (数学库)
- EnTT (ECS)
- Assimp (模型加载)

### 构建步骤
1. 运行 `scripts/Win-GenProjects.bat` 生成Visual Studio项目文件
2. 打开生成的解决方案文件进行编译

## 使用示例

Sandbox目录包含了示例项目，演示了引擎的基本用法：
- Sandbox2D: 2D渲染示例
- TetrisGame: 使用引擎开发的经典俄罗斯方块游戏
